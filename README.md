# SpeakBuddy
🎙️ An AI-powered Android app to practice English speaking with real-time grammar corrections, powered by Grok AI &amp; AWS backend.
SpeakBuddy is an Android application that helps users improve their English speaking skills through AI-driven conversations. Speak into your mic, get an intelligent response from Grok AI, hear it read back via Text-to-Speech, and receive instant grammar corrections — all in one seamless flow. Supports practice modes for Daily Conversation, Job Interviews, Presentations, and IELTS Speaking.
